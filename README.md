# PaidOFF Web (standalone)

This is the standalone frontend (Vite + React + Tailwind) ready for Vercel.

## Quickstart
```bash
npm i
npm run dev
```

## Deploy on Vercel
1) Create a new GitHub repo (e.g., paidoff-web)
2) Push this folder to it
3) Import the repo in Vercel (Framework: Vite)
4) Deploy
